学了这么久的nodejs 都是在本地部署的，今天正好有个朋友给了我一个linux服务器，于是我便尝试在这个服务器上部署node.js, 折腾了大半天时间，前前后后踩了N多个坑，总结了一个最快捷的安装方法

如何在cento下安装Node.js环境

  1. 首先你得有个服务器

  2. 安装SecureCRT 工具（自行百度下载）

  3. 使用SecureCRT 连接服务器

  4. 在SecureCRT命令行输入curl https://raw.githubusercontent.com/creationix/nvm/v0.13.1/install.sh | bash下载nvm 管理工具

  5. 安装成功后输入 nvm list-remote 列出所有可安装的nodejs 版本

  6. 选择一个版本安装 例如 nvm install v5.9.0

  7. 安装成功后输入 node –version 便能显示当前nodejs版本信息了

  8. 输入 node 便能进入node命令行了
